# CFY
# to use the application:
1. install dependencies
```shell
npm install
```
2. build the project
```shell
npm run build
```
3. Run the project
```
npm run start
```

---

### you can use an alternative .env file
### located in  ./.env.examples/production/.env
###  replace .env by this alternative file
